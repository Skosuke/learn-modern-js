# モダンJavaScript学習環境

モダンなJavaScript開発のための学習環境です。

## セットアップ

依存パッケージのインストール:

```bash
npm install
```

## 使い方

開発サーバーの起動:

```bash
npm run dev
```

ビルド:

```bash
npm run build
```

テスト実行:

```bash
npm run test
```

リントとフォーマット:

```bash
npm run lint
npm run format
```

## 注意点

Node.jsのバージョンは`.nvmrc`で指定されています（22.9.0）。
